Analisis Strategi Manajemen Kualitas Data Sosial Stunting di Puskesmas DKI Jakarta

Topik: Manajemen Kualitas Data Kesehatan & Penurunan Stunting Wilayah Studi: Provinsi DKI Jakarta (Berdasarkan Data SSGI 2022) Peneliti: Sutan Faiz Rasyid & Intan Giri Anjani 

ğŸ“ Deskripsi Proyek
Proyek penelitian ini menganalisis strategi manajemen kualitas data sosial stunting di Puskesmas wilayah DKI Jakarta dengan menggunakan konteks epidemiologis dari data Survei Status Gizi Indonesia (SSGI) 2022. Penelitian ini bertujuan untuk memperkuat manajemen kualitas data di tingkat pelayanan primer guna memastikan intervensi penurunan stunting yang tepat sasaran, seperti promosi ASI eksklusif, MPASI, dan PMT.


ğŸ“Š Temuan Utama (SSGI 2022)
Berdasarkan data SSGI 2022, rata-rata prevalensi stunting di DKI Jakarta adalah 14,8%. Terdapat variasi signifikan antar wilayah:
- Tertinggi: Kepulauan Seribu (20,6%) dan Jakarta Utara (18,5%).
- Terendah: Jakarta Selatan (11,9%).

Penelitian ini mengevaluasi kualitas data melalui tiga dimensi utama:
- Sumber Daya Manusia (SDM): Fokus pada peningkatan kapasitas petugas (tenaga gizi, bidan, kader) dalam pengukuran antropometri dan literasi digital.
- Proses: Penerapan prosedur Data Quality Assessment (DQA) secara rutin untuk memastikan kelengkapan, konsistensi, dan validitas data.
- Teknologi: Integrasi sistem informasi kesehatan melalui platform seperti SIGA, SATUSEHAT, dan e-PPGBM untuk pelaporan real-time.

- Jakarta Beraksi: Program kolaborasi lintas sektor yang berkontribusi menurunkan prevalensi stunting secara signifikan.

- Stranas P3S: Integrasi data SSGI ke dalam Strategi Nasional Percepatan Penurunan Stunting 2025-2029.

- Metode: Studi literatur naratif dengan analisis deskriptif kualitatif.
